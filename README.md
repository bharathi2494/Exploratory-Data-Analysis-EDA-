# EDA Practice - Housing Data 
## Project Overview
This notebook performs Exploratory Data Analysis (EDA) on a housing dataset with 545 properties and 13 features including price, area, bedrooms, and various amenities.

## Analysis Process
- Data Loading & Inspection - Initial exploration of dataset structure
- Data Cleaning - Handling duplicates and optimizing data types
- Statistical Analysis - Descriptive statistics and distribution analysis
- Data Visualization - Box plots and categorical analysis
- Data Preparation - Feature selection and type conversion

## Visualizations Included
- Box plot for price distribution
- Value counts for categorical variables
- Statistical summaries for numerical features
